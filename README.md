# Animal Image Gallery

A sleek, responsive mini-portfolio showcasing six high-quality animal images sourced from Pexels. Built for clean presentation, smooth user experience, and simple project structure, this gallery highlights front-end fundamentals and visual design.

---

## Features
- Fully responsive layout  
- Grid-based image display  
- High-resolution photos via Pexels links  
- Lightweight, fast-loading design  
- Beginner-friendly HTML/CSS structure
